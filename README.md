# 🛒 Carrinho de Compra

Sistema simples de **Carrinho de Compras** desenvolvido com foco em praticar conceitos fundamentais de **JavaScript**, manipulação de DOM e lógica de programação.

O projeto simula o funcionamento básico de um e-commerce, permitindo ao usuário visualizar produtos, adicionar itens ao carrinho, remover produtos e acompanhar o valor total da compra em tempo real.

---

## 📌 Objetivo do Projeto

Este projeto foi criado com o intuito de reforçar conhecimentos em:

- Manipulação de elementos HTML com JavaScript
- Eventos de clique
- Atualização dinâmica de informações na tela
- Estruturas condicionais e repetição
- Organização de código Front-End
- Simulação de regras de negócio de um carrinho virtual

---

## 🚀 Funcionalidades

✅ Listagem de produtos disponíveis  
✅ Adicionar produtos ao carrinho  
✅ Remover produtos do carrinho  
✅ Atualização automática da quantidade de itens  
✅ Cálculo automático do valor total da compra  
✅ Interface simples e intuitiva  
✅ Simulação de experiência de compra online

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5** → Estrutura da página
- **CSS3** → Estilização e layout
- **JavaScript** → Funcionalidades e regras de negócio

## 💻 Como Executar o Projeto

1. Clone o repositório

```bash
git clone https://github.com/JoseFelisbino/carrinho-compra.git
```
2. Acesse a pasta do projeto

```bash
cd carrinho-compra
```

3. Execute o arquivo HTML

Basta abrir o arquivo index.html em seu navegador.
Ou utilize a extensão Live Server no VS Code para melhor visualização.

---

## 🛒 Como Funciona

O sistema apresenta produtos na tela e permite que o usuário:

- Clique em Adicionar ao Carrinho
- Visualize os itens adicionados
- Remova produtos caso deseje
- Veja o valor total sendo atualizado automaticamente

Toda a lógica é feita dinamicamente utilizando JavaScript, sem necessidade de backend.
